## How to create your custom classes

Users are free to create their own custom classes. You may create a new file (you can name the file whatever you want; let us say that you have named `src/utils/userMenuClasses.ts` file) and add the following pattern code in the `userMenuClasses.ts` file. You need to copy the code from the `src/@menu/utils/utilityClasses.ts` file and paste it in the newly created file. You can add your custom classes in the folloing pattern:

```ts title="src/utils/userMenuClasses.ts"
export const verticalNavClasses = {
  customClass: 'custom-class'
}
```

## How to use your custom classes

You need to import the custom classes in the file where you want to use them in the following way:

```tsx
import { verticalNavClasses } from 'src/utils/userMenuClasses'
;<MenuItem className={verticalNavClasses.customClass}>Demo</MenuItem>
```

That's it!
