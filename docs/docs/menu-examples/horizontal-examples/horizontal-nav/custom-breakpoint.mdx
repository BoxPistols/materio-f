# Custom Breakpoint

The `customBreakpoint` prop is used to set a custom breakpoint value for the navigation menu which will override the default breakpoint value (lg screen size - 1200px). This is useful when you want a custom breakpoint value and not from the default breakpoint values given [here](/docs/menu-examples/horizontal-examples/horizontal-nav/breakpoint).

### Props

```ts noCopy
customBreakpoint?: string
```
