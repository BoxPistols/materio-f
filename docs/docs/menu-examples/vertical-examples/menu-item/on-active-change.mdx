# On Active Changes

The `onActiveChange` event is triggered when the active state of the menu item changes.

### Props

```ts noCopy
onActiveChange?: (active: boolean) => void
```

This example demonstrates the `onActiveChange` event with a boolean parameter. When the active state of the menu item changes, the `onActiveChange` event is fired, and the provided function is called with a boolean parameter indicating the new active state.
