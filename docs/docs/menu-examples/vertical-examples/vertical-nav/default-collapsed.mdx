# Default Collapsed

The `defaultCollapsed` prop allows you to set the default state of the navigation sidebar to be collapsed. In this state, the width of the navigation sidebar becomes smaller, and the user will need to hover over the navigation sidebar to expand and reveal the whole menu.

### Props

```ts noCopy
defaultCollapsed?: boolean
```

```ts noCopy title='Default Value'
defaultCollapsed={false}
```
