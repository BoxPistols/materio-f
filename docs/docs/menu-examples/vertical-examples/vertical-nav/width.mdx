# Width

The `width` prop allows you to customize the width of the navigation sidebar according to your needs. The prop can accept either a number or a string as its value. If a number is provided, it represents the width in pixels. If a string is provided, you have the freedom to use any valid CSS value for the width.

### Props

```ts noCopy
width?: number | string
```

```ts noCopy title='Default Value'
width='260px'
```
