# Collapsed + Hover

The `defaultCollapsed` prop is used to set the menu to be collapsed by default, and it will expand on hover.

### Props

```ts noCopy
defaultCollapsed?: boolean
```

```ts noCopy title='Default Value'
defaultCollapsed={false}
```
