# Popout When Collapsed

When the navigation sidebar is collapsed, the `popoutCollapsed` prop allows submenus to pop out and be displayed on hover or click. This is useful when the menu is collapsed and you want to display the submenu items. If you want to toggle the submenu items on click (default is hover), please refer to [this](/docs/menu-examples/vertical-examples/menu/trigger-popout) example.

### Props

```ts noCopy
popoutWhenCollapsed?: boolean
```

```ts noCopy title='Default Value'
defaultCollapsed={false}
```
