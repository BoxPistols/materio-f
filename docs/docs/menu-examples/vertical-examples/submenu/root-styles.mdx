# Root Styles

The Root Style feature provides the flexibility to apply custom styling to the submenu. With the `rootStyles` prop, you can pass a set of CSS properties to customize the appearance of the submenu according to your design preferences. The `rootStyles` prop accepts a CSSObject from the [Emotion](https://emotion.sh/docs/introduction) library.

### Props

```ts noCopy
type RootStylesType = {
  rootStyles?: CSSObject
}
```
