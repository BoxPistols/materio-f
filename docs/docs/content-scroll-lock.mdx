# Content Scroll Lock

## Overview

We have locked the body scroll when the navigation sidebar (menu) is toggled. In other words, when the navigation sidebar is opened in smaller screns, you will not be able to scroll the body content.

## How to remove the lock?

To remove the lock, you need to change the following in the `src/components/layout/vertical/NavToggle.tsx` file:

```diff
- import { useEffect } from 'react'
- import { useLockBodyScroll, useToggle } from 'react-use'

- const [scrollLocked, toggleScrollLocked] = useToggle(false)
- const { toggleVerticalNav, isBreakpointReached, isToggled } = useVerticalNav()
+ const { toggleVerticalNav, isBreakpointReached } = useVerticalNav()

- useLockBodyScroll(scrollLocked)

- useEffect(() => {
-   isToggled ? toggleScrollLocked(true) : toggleScrollLocked(false)
-   // eslint-disable-next-line react-hooks/exhaustive-deps
- }, [isToggled])
```

That's it! Now you can scroll the body content even when the navigation sidebar is toggled.
